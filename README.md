# 우송대학교 메타버스 프로그램
##### • 개발 기간 : 2021.11 ~ 2022.02
##### • 한 줄 소개 : 우송대학교 메타버스 학교 정보 제공 프로그램
##### • Skills : Unity, Python, C#, Agora, ASP.net, Sqlite
##### • 팀 구성 : 5명

## 📌 개발 목적
##### • 2년동안 코로나19로 인하여 학교의 대면 프로그램을 접해보지 못하고, 학교에 대한 정보를 모르는 학생들이 학교에 잘 적응할 수 있도록 도와주는 프로그램  우송대학교 메타버스 서비스  WeGo 입니다.

## 🖥️ 담당한 기능
##### • 캐릭터 움직임 기능
##### • 멀티 플레이 기능
##### • 채팅 및 화상채팅 기능
##### • 퀘스트 기능
##### • NPC 기능
##### • OX 퀴즈 기능
##### • 게시판 기능
